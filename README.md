# wdd430-handcrafted-haven

**Handcrafted Haven** is a web-based marketplace that connects local artisans and crafters with consumers who value handmade goods. Our goal is to empower creators by providing a sustainable platform where they can showcase their skills and products.

## Project Purpose

To build a platform that supports local artisans by leveraging software to create a sustainable environment for their craft and commerce.

## Project Requirements

- Sellers can create and manage their own profiles, showcasing their background and skills.
- Sellers can list products with descriptions, images, and pricing.
- Users can leave reviews and ratings on products to help others discover quality craftsmanship.
